Date: 2002-07-07
Subject: emacs: winner-mode と windmove
Distribution: global

winner-mode ってのが emacs に付いている。
window configuration の変更の undo と redo ができる。
M-x winner-mode で起動しておいて、
undo したい時に C-x <left> でできる。

どんな人が使うのだろう？
Emacs のフレームを目一杯大きくして
その中でウィンドウを細かく分割している人には
window configuration がひょんなことから崩れた時に
undo できると便利かもしれない。

winner-mode を info で調べて windmove とやらを発見した。
これまた以前にこんなのあると面白いかもと思ったものだった。
とりあえず、.emacs.el に追加してみる。

(define-key global-map "\M-N" 'windmove-down)
(define-key global-map "\M-P" 'windmove-up)
(define-key global-map "\M-B" 'windmove-left)
(define-key global-map "\M-F" 'windmove-right)

lookup を使うと window が縦3分割になるので、
その時に down と up が役に立つかも。

(eval-after-load "tex-mode"
'(defun tex-display-shell ()
  "Make the TeX shell buffer visible in a window."
  (require 'compile)
  (compilation-set-window-height
   (display-buffer (tex-shell-buf)))
  (tex-recenter-output-buffer nil))
)

Date: 2002-04-21
Subject: emacs: mapcar*
Distribution: global

elisp の info の mapcar のページで見つけた便利な関数。

(defun mapcar* (function &rest args)
  "Apply FUNCTION to successive cars of all ARGS.
          Return the list of results."
  ;; リストをつくしていなければ
  (if (not (memq 'nil args))
      ;; CARに関数を適用する
      (cons (apply function (mapcar 'car args))
	    (apply 'mapcar* function
		   ;; Recurse for rest of elements.
		   (mapcar 'cdr args)))))

(mapcar* 'cons '(a b c) '(1 2 3 4))
  => ((a . 1) (b . 2) (c . 3))
